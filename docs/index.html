SecOps Automation Workbench

A small .NET workbench for synthetic alert normalization, ATT&CK-style triage, safe playbook recommendations, and deterministic tests.

This is an engineering workbench, not a production SOAR/SIEM/EDR. All data is synthetic and every response step is a dry-run recommendation.

Sample output

These static examples are generated by the CLI from the bundled synthetic alert:

  • Triage report (HTML)
  • Triage report (Markdown)
  • Triage report (JSON)
  • Analyst case note (Markdown)

How it works

The pipeline normalizes an alert, maps observables to ATT&CK-style techniques, attaches synthetic enrichment, selects a safe playbook, and renders a report or case note. See demo.md for a reproducible run and architecture.md for the design.
